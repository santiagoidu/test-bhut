# Trabalho de Programação WEB
Desenvolvimento de uma página WEB. Que utilizando Typescript, axios e os componentes do Bulma Framework, lista de carros consumindo api.<br/>

<Strong>API: </Strong>(http://api-test.bhut.com.br:3000/api/cars)<br/>
<Strong>Bulma: </Strong>(https://bulma.io/)<br/><br/>

# Instalação e execução local
```
npm install
```
```
npm start
```
